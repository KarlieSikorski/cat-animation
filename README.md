# 🐱 Kawaii Cat Animation

A fun little Python project that generates a **cartoony kawaii cat animation** as a GIF using the [Pillow](https://pillow.readthedocs.io/) library.  
The cat features:
- Blinking eyes 👀
- Bouncing movement ✨
- Blushy cheeks 🌸
- Cute paws 🐾
- A floating heart 💕
- Wagging tail 🐈
